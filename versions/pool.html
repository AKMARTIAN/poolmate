epMotion Equimolar Pooling Helper

Pools sequencing libraries to equal moles and writes the Eppendorf epMotion DNA.csv / Buffer.csv worklists. Runs entirely in your browser — nothing is uploaded. Your work auto-saves locally.

Run parameters (all optional — fill in later if you like)
Samples — record names + concentrations first; size optional
#Name / number Conc (ng/µL) Size (bp)
About the numbers

Molarity: nM = conc(ng/µL) × 10⁶ / (660 × size_bp) (660 g/mol per dsDNA base pair; 1 nM = 1 fmol/µL). Each sample contributes equal moles: fmol/sample = target_nM × volume / n, sample_µL = fmol/sample ÷ molarity; buffer tops up to target. Presets are vendor starting points (Illumina 2–4 nM pooling stock; ONT SQK-LSK114/R10.4.1 by fmol) — confirm against your own SOP. Downloaded files are prefixed with a unique pool-concentration + timestamp name so runs never collide in your Downloads folder.
